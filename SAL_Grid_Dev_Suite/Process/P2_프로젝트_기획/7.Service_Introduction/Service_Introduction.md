# P2-7. Service Introduction — Vibe Coding OS (VCOS)

> 작성일: 2026-03-16 | 버전: 1.0

---

## 서비스 한 줄 소개

> **"아이디어를 완성 코드로 자동 변환하는 AI 개발 운영체제"**

---

## 서비스 개요

VCOS(Vibe Coding OS)는 Claude Code와 함께 소프트웨어를 만들고 싶은 사람을 위한 **7단계 구조화 플랫폼**입니다.

복잡한 프롬프트 작성 없이 **7개 화면을 순서대로 채우면**, Claude Code에 바로 붙여넣을 수 있는 완성 명령어가 자동 생성됩니다.

---

## 핵심 기능 요약

### Phase 1 — 빌드 (7단계 순차 워크플로우)

| 단계 | 화면 | 기능 |
|------|------|------|
| 1 | 프로젝트 인텐트 | 프로젝트명, 목표, 기술 스택 입력 |
| 2 | AI 협업 준비 | 핵심 문제 정의, 맥락 기록, 피드백 루프 계획 |
| 3 | 복잡도 분석 | UI·Backend·AI·통합·데이터·보안 6축 자동 계산 |
| 4 | 분대 편성 | HQ + 최대 5개 분대, 총 66명 AI 에이전트 자동 배치 |
| 5 | 스킬 전략 | 25개 스킬 중 자동 선택 → Phase 1/2 분류 |
| 6 | 토큰 최적화 | 3계층 모델 정책으로 비용 최소화 |
| 7 | 명령어 생성 | 모든 데이터 통합 → Claude Code 완성 명령어 |

### Phase 2 — 고도화 (5개 전문 엔진)

| 엔진 | 화면 | 기능 |
|------|------|------|
| 고도화 엔진 | 07-refinement | Phase 2 실행 순서 + 5기준 평가 |
| SAL Grid | 08-sal-grid | 3D 좌표계 프로젝트 관리 |
| 디버그 루프 | 09-debug-loop | 5회 반복 디버깅 자동화 |
| 슬라이드쇼 | 10-slideshow | PPT 슬라이드쇼 웹페이지 생성 |
| 유튜브 | 11-youtube | 유튜브 영상 올인원 자동화 |

---

## 사용 흐름

```
사용자 아이디어
      ↓
[VCOS 7단계 입력]
      ↓
완성 명령어 복사
      ↓
Claude Code에 붙여넣기
      ↓
AI가 코드 완성
      ↓
(필요 시) Phase 2 고도화 엔진 실행
```

## 타겟 사용자

- AI로 처음 코딩해보는 사람
- 구조적으로 AI를 활용하고 싶은 개발자
- 혼자 앱/웹을 만들고 싶은 창업가·기획자
