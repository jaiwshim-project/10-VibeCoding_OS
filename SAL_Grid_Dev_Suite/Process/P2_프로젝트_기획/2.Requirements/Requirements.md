| ID | 요구사항 | 기준 |

## 3. 제약 조건

1. **외부 프레임워크 금지** — 순수 HTML/CSS/JS 유지
2. **서버 불필요** — 정적 파일만으로 동작
3. **localStorage 의존** — 브라우저 스토리지 기반 상태 관리
4. **Claude Code 의존** — AI 실행은 사용자의 Claude Code 환경에서
