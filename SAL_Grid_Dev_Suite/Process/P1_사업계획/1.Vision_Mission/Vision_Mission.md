# P1-1. Vision & Mission — Vibe Coding OS (VCOS)

> 작성일: 2026-03-16 | 버전: 1.0 | 소급도입 (기존 프로토타입 기반)

---

## Vision (비전)

> **"아이디어를 가진 모든 사람이 소프트웨어를 만들 수 있는 세상"**

코딩 지식 없이도 AI와 협력하여 완성도 높은 소프트웨어를 만드는 것이 일상이 되는 미래를 만든다.

---

## Mission (미션)

> **"AI 시대의 개발 방법론을 체계화하고, 누구나 따라할 수 있는 운영체제처럼 제공한다"**

- AI와 인간이 협력하는 최적의 개발 프로세스를 정의하고
- 7단계 구조화된 워크플로우로 아이디어를 실제 동작하는 코드로 변환하며
- 25가지 전문 스킬과 66인 AI 에이전트 시스템으로 엔터프라이즈급 품질을 민주화한다

## Elevator Pitch (30초 소개)

> "AI로 코딩하고 싶은데 어떻게 해야 할지 모르겠다는 분들을 위해 만들었습니다.
> VCOS는 아이디어를 입력하면 AI가 분석하고, 팀을 구성하고, 완성 코드를 만들어주는
> Vibe Coding을 위한 운영체제입니다. 7단계를 따라가기만 하면 됩니다."

---

## 프로젝트 배경

### 문제 인식
- **AI 코딩 도구의 보급** → 누구나 시도하지만, 구조 없이는 실패율이 높음
- **"Vibe Coding"의 한계** → 즉흥적 프롬프팅은 일관성이 없고 품질 불균일
- **전문가-비전문가 격차** → 개발 경험이 없으면 AI에게 올바른 지시를 내리기 어려움

### 해결책
VCOS는 AI 개발의 **베스트 프랙티스를 운영체제(OS)처럼 추상화**하여,
사용자가 **단계만 따라가면** 고품질 소프트웨어가 나오는 시스템을 제공한다.

- KR1: 7단계 워크플로우 전체 동작 (01-intent → 07-command)
- KR2: Phase 2 고도화 5개 페이지 안정화
- KR3: 25개 스킬 전체 문서화 + Skills Map 완성

### Objective 2: 방법론 확장
- KR1: SAL Grid Dev Suite 소급도입으로 VCOS 자체 품질 관리 시작
- KR2: VCOS를 사용하여 VCOS를 개발하는 "자기참조" 달성
