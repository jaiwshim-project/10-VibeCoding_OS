# P1-3. Business Model — Vibe Coding OS (VCOS)

> 작성일: 2026-03-16 | 버전: 1.0

---

## 1. 비즈니스 모델 캔버스

### 가치 제안 (Value Proposition)

| 대상 | 핵심 가치 |
|------|---------|
| AI 코딩 입문자 | 7단계 가이드로 어떤 프로젝트든 시작할 수 있는 자신감 |
| 솔로 개발자 | 66명 AI 에이전트 팀의 효과 — 혼자이지만 팀처럼 |
| 스타트업 팀 | 엔터프라이즈급 방법론을 소규모 팀 비용으로 |

---

## 2. 수익 모델 (Revenue Streams)

### Phase 1 (현재): 오픈소스 / 자기 사용
- VCOS 플랫폼: **완전 무료** (GitHub Pages)
- 스킬 패키지: **개인 사용 목적** (SAL Grid, AX덴탈 등)
- **목표**: 플랫폼 완성도 확보 + 방법론 검증

### Phase 2 (6개월 후): 스킬 마켓플레이스
```
┌─────────────────────────────────────────────────────────────┐
│  Free Tier                                                  │
│  - VCOS 플랫폼 전체 무료                                     │
│  - 기본 스킬 5개 무료                                        │
│  - SAL Grid Dev Suite Lite                                  │
├─────────────────────────────────────────────────────────────┤
│  Pro Tier (~$29/월)                                         │
│  - 25개 전체 스킬                                            │
│  - 커스텀 스킬 생성기                                        │
│  - SAL Grid Dev Suite Full                                  │
│  - 우선 지원                                                 │
├─────────────────────────────────────────────────────────────┤
│  Enterprise                                                 │
│  - 팀 라이선스                                               │
│  - 커스텀 스킬 개발                                          │
│  - 방법론 컨설팅                                             │
└─────────────────────────────────────────────────────────────┘
```

### Phase 3 (1년 후): 교육 & 컨설팅
- **온라인 강의**: "AI로 만드는 실전 프로젝트" 시리즈
- **방법론 컨설팅**: 기업 AI 개발 체계 구축
- **SAL Grid 라이선스**: 기업 내 도입 지원

## 3. 핵심 자원 (Key Resources)

| 자원 | 설명 |
|------|------|
| **방법론 IP** | SAL Grid Dev Suite V3.2 — 핵심 경쟁력 |

## 4. 채널 (Channels)

1. **GitHub** — 오픈소스 배포, 개발자 커뮤니티
2. **YouTube** — 사용법 튜토리얼 (VCOS 11-youtube.html 활용)
3. **SNS** — LinkedIn, X(Twitter), 개발자 커뮤니티
4. **Word of Mouth** — Claude Code 사용자 간 추천

## 5. 비용 구조 (Cost Structure)

| 항목 | 현재 비용 | 규모화 시 |
|------|----------|---------|
| 서버 비용 | $0 (GitHub Pages) | $0 유지 (정적) |
| AI API 비용 | Anthropic Claude 구독 | 사용자별 개인 부담 |
| 개발 인력 | 1인 (Sunny) | 필요 시 프리랜서 |
| 마케팅 | $0 (유기적 성장) | 성장 후 투자 |

**핵심 장점**: 정적 HTML 플랫폼으로 서버 비용 $0
